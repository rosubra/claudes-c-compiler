Implement SSE2 intrinsic builtins needed for PostgreSQL build.

PostgreSQL's wchar_srv.c includes <emmintrin.h> which uses these __builtin_ia32_* functions:
- __builtin_ia32_movnti64(long long *, long long) - non-temporal 64-bit store (movnti)
- __builtin_ia32_movntdq(__v2di *, __v2di) - non-temporal 128-bit store (movntdq)
- __builtin_ia32_movntpd(double *, __v2df) - non-temporal double-pair store (movntpd)
- __builtin_ia32_clflush(void const *) - cache line flush (clflush)
- __builtin_ia32_lfence(void) - load fence (lfence)
- __builtin_ia32_mfence(void) - memory fence (mfence)

Plan:
1. Register these in sema/builtins.rs as intrinsics
2. Lower them in expr.rs to Instruction::InlineAsm with appropriate x86 templates
3. Add test case
