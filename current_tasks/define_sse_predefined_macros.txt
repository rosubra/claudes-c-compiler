Define __SSE__, __SSE2__, __MMX__ and related SIMD predefined macros for x86/x86_64 targets.

GCC/Clang define these macros by default:
- x86_64: __SSE__, __SSE2__, __MMX__, __SSE_MATH__, __SSE2_MATH__ (SSE2 is baseline for x86_64)
- i686: __SSE__, __SSE2__, __MMX__, __SSE_MATH__, __SSE2_MATH__ (default for modern GCC -march=i686)

These are needed by projects like dr_libs, minimp3, stb_image, and many others that use
#ifdef __SSE2__ to conditionally enable SIMD code paths.

The macros should NOT be defined when -mno-sse is passed.
