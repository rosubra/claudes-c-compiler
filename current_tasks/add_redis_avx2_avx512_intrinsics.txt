Task: Add AVX2 and AVX-512 intrinsic headers for Redis compilation
Status: in_progress

Goal: Enable Redis to compile by adding the specific AVX2/AVX-512 intrinsics
that hyperloglog.c and bitops.c need.

Files to create:
1. include/avxintrin.h - __m256, __m256d types and basic AVX load/store
2. include/avx2intrin.h - __m256i integer intrinsics (and, or, shift, shuffle, etc.)
3. include/avx512fintrin.h - __m512i type and minimal AVX-512 intrinsics
4. Update include/immintrin.h to include new headers

Specific intrinsics needed by Redis:
- __m256i type
- _mm256_loadu_si256, _mm256_storeu_si256, _mm256_lddqu_si256
- _mm256_and_si256, _mm256_andnot_si256, _mm256_or_si256, _mm256_xor_si256
- _mm256_set1_epi32, _mm256_set1_epi64x, _mm256_setr_epi8
- _mm256_shuffle_epi8
- _mm256_slli_epi32, _mm256_srli_epi32
- _mm256_max_epu8
- _mm256_castsi256_si128, _mm256_extracti128_si256
- __m512i type
- _mm512_loadu_si512, _mm512_popcnt_epi64, _mm512_reduce_add_epi64
