Fix ARM builtin assembler missing instructions and linker local symbol resolution.

Issues fixed:
1. ARM assembler: missing ldadd/ldaddal/ldclr/ldeor/ldset (LSE atomics) instructions
2. ARM assembler: missing sri (Shift Right and Insert) NEON instruction
3. ARM assembler: missing dup Vd.T, Vn.Ts[i] (element-to-all-lanes) form
4. ARM assembler: wrong TLS relocation type (R_AARCH64_TLSLE_ADD_TPREL_LO12 vs _NC)
5. ARM linker: local symbols incorrectly resolved via globals table
   (e.g., static function named "write" resolved to libc write() instead of local function)
